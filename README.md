# Core Utils
## _A simple unity package containing C# Utils classes_

This is a simple library I am maintaining with all the code utils I have used, added, created, or stole from around the web. Feel free to use it as well!

## Features
### Runtime

- Extensions files for Arrays, GameObjects, Lists, Math, Span and Vectors
- A simple FSM
- A GameUpdater so you don't have to rely on Unity Update methods
- A simple, static Messaging System
- A Pooling System
- A Service Locator
- Singleton Pattern
- Spatial Partitionning (KDTree, Quadtree and Octree)

### Editor

- EnumProperty Drawer
- InterfaceProperty Drawer
- ReadOnly Attribute
- ResetOnPlay Attribute, to help with disabling Domain Reload
